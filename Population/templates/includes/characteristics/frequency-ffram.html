Framework Front-End

Los frameworks Front-End son herramientas que facilitan el diseño e interactúan como Plugins para la dinámica del DOM (Document Object Model), estas herramientas pueden ser hojas de estilos, librerías JavaScript o la combinación entre hojas de estilos CSS y JavaScript. En la gráfica se muestra las frecuencias de uso de Frameworks Front-End en el desarrollo de aplicaciones web.
